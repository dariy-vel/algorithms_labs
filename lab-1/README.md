# Algorithms lab-1

I have made my own realisation of Selection sort and Merge sort in Python3

## How to run it
If you want to run this on your machine, read the following instructions
#### Prerequisites

You'll need only Python3 installed

### Installing

Download this repository as zip and unpack it to empty folder or clone it to your local storage.
Then open terminal in folder containing `main.py`.
* For Windows run ```python main.py```
* For Linux run ```python3 main.py```

Also feel free to modify `file.txt` and add more helicopters
